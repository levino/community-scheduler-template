# Welcome to Community Service Scheduler

A modern template for organizing community service schedules using GitHub, AI, and static site generation.

## Features

- **No backend needed** - GitHub as your admin interface
- **AI-powered** - Claude Code assists with updates
- **Mobile-friendly** - Access schedules from any device
- **Type-safe** - Validated data with build-time checks

## Quick Access

- [Documentation](/docs/intro) - Learn how it works
- [Contact Persons](/docs/ansprechpartnerinnen) - View all contacts
- [Statistics](/dienstplan/auswertung) - Service completion overview

## Get Started

1. Fork this template
2. Customize your data in `src/content/`
3. Deploy to GitHub Pages, Cloudflare, Netlify, or Vercel
4. Start managing schedules!
