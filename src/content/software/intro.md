# Community Scheduler Template

Willkommen zur Dokumentation des **Community Scheduler Templates** - einer flexiblen Lösung für die Verwaltung von wiederkehrenden Gemeinschaftsaufgaben.

---

## Was ist das Community Scheduler Template?

Das Community Scheduler Template ist ein **Open-Source-Projekt**, das die Koordination von regelmäßigen Diensten in Gemeinschaften vereinfacht:

- **Kitas & Kindergärten** - Eltern-Kochdienste, Putzdienste
- **Schulen** - Elternabende, Aufsichtsdienste
- **Vereine** - Schichtpläne, Ehrenamtliche Dienste
- **Nachbarschaften** - Gemeinsame Gartenarbeit, Hausmeisterdienste
- **Büros** - Küchendienste, Meetingraum-Verwaltung

---

## Kernfunktionen

### Automatische Validierung
Fehlerhafte Daten werden vor der Veröffentlichung abgelehnt - keine Tippfehler, kein falsches Datum.

### SMS-Erinnerungen
Automatische Benachrichtigungen vor jedem Dienst.

### Mobilfreundlich
Responsive Design mit direkt anrufbaren Telefonnummern.

### Statistiken
Übersichtliche Auswertung: Wer hat wie oft Dienst gemacht?

### Vollständige Historie
Alle Änderungen werden mit Begründung dokumentiert.

### KI-Unterstützung
Tausch-Anfragen können per KI (Claude) bearbeitet werden.

---

## Erste Schritte

1. **Fork erstellen** - Das [GitHub Repository](https://github.com/levino/community-scheduler-template) forken
2. **Anpassen** - Namen, Termine und Texte für Ihre Gemeinschaft anpassen
3. **Deployen** - Auf GitHub Pages, Cloudflare Pages oder anderen Plattformen veröffentlichen

Detaillierte Anleitungen finden Sie in der [README](https://github.com/levino/community-scheduler-template#readme).
